# rohc_enhanced

Robust Header Compression Tests

### How To Use

#### Build With GCC:

```C
gcc -o rohc_enhanced -Wall \
  $(pkg-config rohc libpcap --cflags) \
  rohc_enhanced.c \
  $(pkg-config rohc libpcap --libs)

```
